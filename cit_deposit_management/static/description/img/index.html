Feature

User can easily check another user environment (if user have access to do that).
This module allows user to see another user environment and easily back to users's original environment.
So it eliminates login procedure.

Configurations

Here we can set restrict impersonate user.

How It Works

After giving rights to user, we can see button there as you can see in above image, using that button user can directly impersonate that specific user and login as that user.
After click impersonate button on user "Marc Demo", Admin login as "Marc Demo" user. and can see all menu as "Marc Demo" have access permission.
User can easily go back to his original environment by using "Close Impersonate" button.
Note: We recommend this module can be used for staging / QA / development environments. Think before using in production
           environment as we are allowing user pretend to be another user.
