Hospital-Healthcare and Clinic Management System Odoo Apps

Hospital management system Odoo Apps helps to manage day to day activities of hospital and clinic which can be helpful for any Hospital and medical facilities. This module helps to manage Patient details,Doctors details, Patient Appointment, Prescription, Patient Hospitalization, Lab Test request and result, medicament, new born details etc. This odoo apps also helps to generate Report for patient details, appointment, prescriptions, lab-test

Features

Manage Patient details

Patient can enter their details.There are tabs will shows patient details like medicament, diseases, lab test, etc.

Receptionist can Create Appointment

Receptionist can create appointment with the details like physician , consultation service etc.

Manage New Born details

Receptionist can create appointment with the details like physician , consultation service etc.

Generate Patient's Prescriptions

Generate prescriptions with the doctor's and patient's details, can create invoice from prescription.

Lab Test Request and Result

Patient can request for the lab test and can get the result of the lab test.

Generate Report for patient details, appointment, prescriptions, lab-test

User can generate the report for their details, appointment, prescriptions, lab-test.
Demo Request Video Tutorial

Hospital Management Menu

User can see added new main menu for hospital management.

User Access

  • Chemist
  • Doctor
  • Medical Group
  • Receptionist

Users with particular access can manage and see documents according to role.


Hospital Management Configuration

In Hospital Management > Configuration menu users can see and configure different things for the hospital like lab test unit and type, pathology group, diseases, diseases categories, imaging test and type, medical procedures, Health center, Building, ward, unit, bed, physician, medicament, insurance and etc.

Patients

In Hospital Management > Patients > Patients menu users can create patients and enter all the details of patients, All other tabs will show details about patients like medicament, lab test details, etc.

Patients Card Report

By clicking on the "PRINT PATIENT CARD" button user can print a patient card pdf report.

Patients Medication History Report

Patients Vaccinations History Report

Patients Diseases Report

Patients Lab Test Report

Families

In Hospital Management > Patient > Families menu users can create and enter the family members and their contact details for future reference to contact them in case of emergency.

Prescriptions

In Hospital Management > Patient > Prescription menu user can create a prescription for the patient, user can enter the details of the prescribed medicament, its dosage, and other details.

Prescription Report

Users can print prescription pdf report by clicking the "PRINT PRESCRIPTION" button.

Prescription Invoice

Users can create an invoice for a prescription by clicking the "CREATE INVOICE" button.

Appointments

In Hospital Management > Appointments menu the user can create a new appointment, view and manage an appointment.

Appointments Report

Appointments Invoice

Users can create an invoice for an appointment by clicking the "CREATE INVOICE" button.

Patient Hospitalization

In Hospital Management > Inpatient Administration > Patient Hospitalization menu users can create patient hospitalization and enter detail about his/her hospitalization, patient's bed details, hospitalization date & expected discharge date, and other details and click the "CONFIRM REGISTRATION" button.

Patient Admission

On clicking the "PATIENT ADMISSION" button user can see the patient admitted and the state changed to "Hospitalized" from confirmed.

Hospital Bed Transfer

Users can transfer patient beds from the "Bed Transfer" menu under the "Action" menu.

User need to select a new bed and the reason for bed transfer and click the "TRANSFER" button.

Discharge Patient

Users can see beds added to a patient hospitalized, User also can discharge a patient by clicking the "DISCHARGE PATIENT" button.

Create Imaging Request

In Hospital Management > Imaging > Create Imaging Request menu users can create imaging requests for the patient.

User need to select patient, tests, and other details.

Confirm Imaging

User can confirm imaging requests by clicking the "CONFIRM" button.

Create Imaging Result

User can create imaging results for imaging tests by clicking the "CREATE RESULT" button or from the "Imaging Test Result" menu.

Pediatric Symptoms Checklist

In Hospital Management > Pediatrics > Pediatric Symptoms Checklist menu users can create a symptoms checklist for the child.

Newborn

In Hospital Management > Pediatrics > Newborn menu users can create and manage newborns, and enter all the necessary details about newborns. like main details, APGAR score, reanimation, neonatal signs and symptoms, neonatal reflex check, congenital diseases, medication, and test details.

Newborn Card Report

Lab Requests

In Hospital Management > Laboratory > Draft Lab Request menu users can create lab requests.

Lab Tests Results

In Hospital Management > Laboratory > Lab Test Result menu users can create test results, Also by clicking the "CREATE LAB TEST" button and enter test cases.

Lab Tests Invoice

Users can create lab test invoice by clicking the "CREATE LAB INVOICE" button.

Surgery

User can create surgery and enter a description of surgery, classification of surgery, and other details.
